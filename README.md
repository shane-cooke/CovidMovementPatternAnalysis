# COVID-19 Movement Pattern Analysis

***

##To run project manually:

`python3 MovementPatternAnalysis.py`

##To run project in Jupyter:

`Jupyter Notebook`
Click run.

---

##To run the project using docker:

1) `mvn clean install`

2) `docker compose build`

3) `docker compose up`

4) `mvn exec:java -pl client`
